Menstrual Health & Awareness Website

Live Project Link

This project focuses on spreading awareness about menstrual health through a clean, informative, and user-friendly web platform. It aims to educate users while providing helpful tools in an accessible manner.

The website provides educational content, FAQs, and a basic period tracking interface to help users understand menstrual cycles better. The focus was on clarity, simplicity, and responsive design.

Presenting sensitive health-related information in a simple, respectful, and easy-to-understand format while keeping the interface engaging and responsive across devices.

Designed a clean and minimal UI using HTML and CSS, implemented interactivity using JavaScript, and ensured the site is mobile-friendly and accessible to a wide audience.

Key Features

  • Informative awareness content
  • FAQ section for common queries
  • Simple period tracking UI
  • Responsive and user-friendly design
